# VidSnap — AI 视频理解助手

> **一句话**：丢给它一个 YouTube/抖音链接，它替你看完，然后你可以问任何问题。

## 产品定位

不是泛化的"多模态知识助手"，而是专注解决一个高频刚需——**"视频太长/太水，我只想知道讲了什么"**。

## MVP 核心功能（3个）

| 功能 | 输入 | 输出 |
|------|------|------|
| 📝 视频总结 | 抖音/YouTube 链接 | 结构化摘要 + 分段要点 + 总时长 |
| 🔑 关键信息提取 | 视频链接 + 关注点 | 结构化列表，每项带时间戳 |
| 🌐 跨语言翻译 | 外语视频链接 | 中文字幕 / 中文总结 |

## 技术栈

- **前端**: Next.js
- **视频处理**: yt-dlp（下载）+ Whisper Large v3（ASR）+ FFmpeg（关键帧）
- **AI 引擎**: Claude API (Opus 4.8)
- **任务队列**: BullMQ + Redis
- **存储**: S3 兼容对象存储
